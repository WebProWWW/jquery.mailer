# jQuery Mailer
Sending HTML 	&lt;form&gt; with jQuery ajax
### Quick start
```html
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.2.1/jquery.min.js"></script>
<script src="/path/to/jquery.mailer.min.js"></script>
<script>
  $(function() {
    $('.js-mailer').mailer({
      // Options will go here
    });
  });
</script>
```
